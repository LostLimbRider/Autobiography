# Lost Limb Riders Pamphlet Package

This folder contains the production draft for the Lost Limb Riders handbook.

## Files

- `lost-limb-riders-handbook.md` — full handbook draft with image placeholders and partner-ready language.
- `images/` — placeholder files named to match the image references in the handbook.

## Production Notes

Before final public release, replace bracketed placeholders with approved organizational information, including founder name, contact information, nonprofit details, official photographs, and sponsor information.

The handbook is written in clean Markdown so it can be imported into a layout tool, converted to Word format, or adapted into a PDF design package.
